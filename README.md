# 🧠 Mental Health Tracker

A web-based mental wellness platform designed to support users in managing their emotional well-being through mood tracking, self-care recommendations, counselor support, and progress monitoring.

## 🌟 Features

### 🔐 User Authentication

* Login and signup interface for users.
* Secure access to mental wellness resources.

### 😊 Mood Tracking

* Users can select their current mood.
* Acts as the foundation for personalized recommendations.

### 💡 Personalized Self-Care Suggestions

* Provides self-care activities and wellness tips based on the user's selected mood.
* Encourages healthy coping strategies and positive habits.

### 👩‍⚕️ Counselor Support

* Browse available counselors.
* Explore professional support options.

### 📅 Appointment Booking

* Schedule counseling sessions through the booking module.
* Simplifies access to mental health assistance.

### 💬 Chat Support

* Interactive chat feature for engagement and guidance.
* Creates a supportive and accessible user experience.

### 📊 Progress Tracking

* Monitor emotional well-being over time.
* View progress reports and track personal growth.

## 🛠️ Technologies Used

* Python

* HTML

## 🚀 Getting Started

### Prerequisites

* Python 3.x installed
* Flask installed

### Installation

1. Clone the repository:

```bash
git clone https://github.com/SumaiyyaFatima/Mental-Health-Tracker.git
```

2. Navigate to the project directory:

```bash
cd Mental-Health-Tracker
```

3. Install dependencies:

```bash
pip install flask
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```text
http://127.0.0.1:5000
```
